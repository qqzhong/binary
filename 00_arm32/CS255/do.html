The do-while statement

  • How to realize the do-while statement in assembler code

    • The Java do-while statement has the following syntax: 

           do
   {
      statement(s)
   } while ( condition )

      Any do-while statement can be re-written as follows with a while statement 

           statement(s)

   while ( condition )                        
   {
      statement(s)
   }
