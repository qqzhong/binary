Program flow of a do-statement
  

                                       |
   do                                  |<---------------+
   {                                   |                |
     statement(s)	    	+--------------+        |
   } while (condition)	    	|  statement(s)|        |              
			    	+--------------+	|
			    	       |		|
			    	       |     		|
			    	       V		|(repeat loop)
			        +--------------+	|
			    	|  condition   |------->+
                                +--------------+  (TRUE) 
			    	       | (FALSE)        
				       V

The do-statement is rarely used - so no examples...
