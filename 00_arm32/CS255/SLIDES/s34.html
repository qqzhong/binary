Control structures in High Level Programming Languages
 

Control structures alters the default program flow

Commonly used control structures in high level programming languages:

  • The if-statement
  • The if-else-statement
  • The switch-statement

  • The while-statement
  • The for-statement
  • The do-while-statement

 

I will omit the switch-statement because it's equivalent to nested if-else-statements

I will omit the do-while-statement because it is rarely used in programs

Assembler instructions used to implement control structures in assembler

Any control structure can be implemented using these 2 assembler instructions:

  1. Compare instruction:

      • This instruction compares 2 values and set the status flags (N, Z, V, C) in the PSR according to result of the comparison

  2. Conditional and unconditional branch instructions:

      • This group of instructions can make the CPU "jump" to a given memory location when a certain condition is satisfied.

        If the condition is not satisfied, the CPU will continue execution with the next instruction (i.e.: in-order program flow).
