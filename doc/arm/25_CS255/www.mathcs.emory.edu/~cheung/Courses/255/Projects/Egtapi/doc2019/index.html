Prelude

  • Download page of the Egtapi software: click here

    To install: create a folder (I recommend: "cs255") and unpack the zip file that you have downloaded in the "cs255" folder.

    You can then run EGTAPI by using the file browser/explorer on your laption. Browse in the "cs255" folder and click on the Egtapi icon to run EGTAPI.

  • If you want to try out the features of Egtapi while reading this documentation, then make a copy of the fac.s demo assembler program to your cs255/Egtapi directory by doing the following: 

          1. login to a lab machine

       (If you're home, login to lab0z.mathcs.emory.edu         
        and then to a real lab machine)

    2. Create your own "Egtapi" directory:

          mkdir ~/cs255/Egtapi

    3. Go to the "Egtapi" directory:

          cd ~/cs255/Egtapi

    4. Copy the fac.s assembler program:

          cp ~cs255001/demo/asm/8-sub/fac.s  ~/cs255/Egtapi   

    5. Copy the debug files:

          cp ~cs255001/demo/asm/8-sub/.egt.fac.arm ~/cs255/Egtapi   

    6. Then compile (assemble) the program:

          /home/cs255001/bin/as255 fac 

    7. Run Egtapi on your laptop

       If you're using a Lab machine, execute this in a window: /home/cs255001/bin/egtapi &     

       The Egtapi window will pop up and you are ready to use Egtapi

       (Learn what to do by following the examples in this tutorial)

    Try out Egtapi by pressing the buttons in the Egtapi window and watch the effect as explained in this doc.
