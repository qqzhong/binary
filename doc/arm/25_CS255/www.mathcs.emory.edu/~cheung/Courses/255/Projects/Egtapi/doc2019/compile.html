5. Compiling your assembler programs

  • Fact:

      • An assembler program is compiled from one or more assembler program files

  • Compiling programs that consists of a single assembler source file

    • The EGTAPI window provides a quick way to compile assembler programs that consists of one single program file

      For example:

        • Programs used in the demo in class consists of one single assembler program file

      How to compile a program that consists of one single program file:

        1. Click on File Browser:

        2. Click on the assembler program file (filename must end in .s):

          Click on "Compile" to compile the select assembler program

          (Click File Browser again to hide the File Browser panel after you are done)

  • Compiling programs that consists of multiple assembler source files

    • Fact:

        • Some of the programs for your projects consist of multiple assembler source file

            • Each project/homework assignment will contain a UNIX command on how to compile your project

    • How to compile a program that consists of multiple program files:

        1. Click on Terminal:

          This will bring up the Terminal panel

        2. Check your current directory by executing the pwd command:

        3. If necessary (i.e., if you're not currently in the directory where your project files are stored), use the cd (change directory) command to go to the correct sub-directory where your program files are stored:

        4. Use the ls (list) command to make sure the correct program files are present:

          In the above example, I changed the current directory to /home/cheung/cs255/hw6

        5. Once you're in the correct directory (= the directory where your project files are stored), you can compile your program with the appropriate command given in the homework assignment page:

          In the above example, I ran the command "as255 hw6 bubblesort" that compiles assembler files hw6.s and bubblesort.s into the program pj6
