6. Loading/reloading your assembler program

  • What is "loading" a program

    • Loading a program:

        • Loading a program = reading the instructions of a program into EGTAPI so EGTAPI can execute the instructions of that program

  • Loading a program into EGTAPI

    • How to load a program into EGTAPI for execution:

        1. Click on File Browser:

        2. Click on arrow of the folder (sub-directory) where the program is stored:

        3. Click on program file that you want to run:

        4. Click on the Select button to load the selected program into EGTAPI:

        5. EGTAPI should display that it is stopped at the Start label:

          Now you can run your assembler program

  • Re-loading a program into EGTAPI

    • Re-loading a program:

        • Re-loading a program = load the previously selected program

        • Reloading will load a previously selected program and start running it from the beginning

          This is a quick way to run the same program after you have fixed some bugs and have compiled the updated source file.

    • How to re-load a program into EGTAPI:

        • Click in the Reload button at the top-right corner in the EGTAPI window:
