Declaring a function that uses array parameter variables

  • Syntax used to declare functions

    • Recall the syntax to declare a function: (See: click here) 

           [extern]  returnType  functionName ( param1type, param2type, .... )

  • Declaring a function that has an array parameter variable

    • Just follow the standard syntax

    • To specify a one-dimensional array as parameter variable, use (the "convenient notattion"): 

            datatype [ ]

      Example: 

            [extern]  double sum( double [ ] );

    • To specify a two-dimensional array as parameter variable, use: 

            datatype  [ ] [ SIZE2 ]

      Example: 

            [extern]  double sum( double [ ] [10] );

    • Example: 

           /* --------------------------------------------
      Declare a function with an array parameter
      -------------------------------------------- */         
   extern void func( double [ ] ); 
    
    
   int main( int argc, char* argv[] )
   {
      double a[10];
      int    i;
    
      for ( i = 0; i < 10; i++ )
   	 a[i] = i;
    
      func( a );
   }
    
    
   void func( double x[ ] )
   {
      int i;
    
      for ( i = 0; i < 10; i++ )
   	 printf( "x[%d] = %lf\n", i, x[i] );
   }

  • Postscript

    • You can really make the function declaration look more normal with a variable name that is treated as comment:

      Declare a function with an array parameter
      -------------------------------------------- */         
   extern void func( double x[ ] );     // add a dummy var name !!!

      double a[10];
      int    i;
