The output problem of integer numbers

  • Suppose you have a byte variable x and it contains the 2s complement code 00000011 (= value 3(10) (i.e.: 48 dots))

    When we print x, we want to see: 

          3

  • If we send 00000011 to the terminal, it will print a strange symbol !!!

    (Because 00000011 is an unprintable ASCII code for "End of Message")

  • In order to print the number 3 on a terminal, we must send this ASCII code to the terminal: 

         00110011
