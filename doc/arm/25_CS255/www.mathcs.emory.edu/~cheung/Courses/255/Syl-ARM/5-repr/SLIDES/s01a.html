How do human represent information (codes !!!)

  • Humans is probably the only being that can represent information

  • The way that human represent information is through the use of:

      • codes           

  • Encoding (information):

      • Encoding is a process where some fact is represented by some symbol(s) (= code(s))

  • Decoding (a code):

      • Decoding (a code) is the reverse process where some symbol(s) (= code(s)) is interpreted (to obtain its meaning (= information))
