How do we represent information ?

Consider the following problem:

  • We want to store information of people's gender and marital status in a computer program

    Gender information:   male or female
    Marital status information:   single, married, divorced or widowed

 

But we have this problem:

  • The computer can only store binary numbers
 

How can we represent male/female or single/married/divorced/widowed using (binary) numbers ???

How to represent information inside the computer
 

The principle of information representing is:

  • Code + Context = Information

    I.e.: information is represented by a code that must be interpreted with the correct context

    The correct context must be used in order to extract the correct information

 

 

I will show you some examples on how to represent information next

(1) Example on how to represent "gender" information
 

Example: how to represent "gender" information

  • We can encode the gender information of people as follows: 

          0  =  male
    1  =  female

    I.e.: we agree that when storing "gender" information, we use 0 to represent "male" and 1 to represent "female"

 

Note:   this is an example of the use of codes to present information !!!

(2) Example on how to represent "marital status" information
 

Example: how to represent "marital status" information

  • We can encode the marital status information of people as follows: 

          0  =  single
    1  =  married
    2  =  divorced
    3  =  widowed

    I.e.: we agree that when storing "marital status" information, we use 0 to represent "single", 1 to represent "married", and so on

 

Note:   I used decimal number (0,1,2,3) for simplicity.
Computers will use binary numbers: 00, 01, 10 and 11

Application:   how do computer programs store information ?
 

  • Computer programs store information inside their (program) variables

  • The variables in computer programs can only store (binary) numbers !!!

  • The data type of a variable convey the context used to interpret the value (= number) stored in the variable

  • In fact:

      • A programming language provides many (different) data types

      • Each data type uses its own code !!!

      • So: each data type will have its own way to encode/decode the information !!!

Application:   how do computer programs store information ?
 

A Java program that shows that the data type contains the context to interpret the code 65: 

public class Context
{
   public static void main(String[] args)
   {
      int  x;
      char y;

      x = 65;
      System.out.println(x); // Prints "65"

      y = 65;
      System.out.println(y); // Prints "A"
   }
}
