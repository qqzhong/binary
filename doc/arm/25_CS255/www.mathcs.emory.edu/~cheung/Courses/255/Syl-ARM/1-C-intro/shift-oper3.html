Setting the bit at poistion n in an int variable

  • Setting a bit

    • Recall: how to set a bit 

            Bit position:                                  6543210

    Starting bit pattern:   ..............x...............
    Bit mask:               000000000000001000000000000000
                        ----------------------------------------- OR      
                            ..............1...............

    (Bit dnoted by . will remain unchaned)

  • Making the bit mask

    • The bit mask 00000000000010000000000000... canbe made with the following operation: 

                                   i positions
                       <-------------->
  1 << i   =   000000..1000000000000000

  • C statement to set bit position i in an int variable

    • C code: 

           int x;
   int i;

   x = x | (1 << i );    // Set bit position i in x

  • Macro definition for setbit operation

    • C macro to set bit at position i in a variable x: 

           #define   SETBIT( x, i )     ( x |= (1 << i) )
