How the program stack avoids overwriting parameters and local variables
 

Because of the properties of the stack:

  • A stack will grow when things are pushed on the stack

  • The new things pushed on the stack will not overwrite the old things stored inside the stack !!!

How the program stack avoids overwriting parameters and local variables
