What and why are data structures in a program ?

  • Computer programs always work with data (= information)

    Often with a large amount of data !!!

  • Large amount of data (= information) must be organized to we can find the needed information item quickly !

  • The structures used to organizes data in a computer program is called:

      • Data structures        

  • The 2 most commonly used data structures are:

      1. Arrays
      2. (Linked) lists        

Simple and composite variables

  • Simple (or "unstructured") variables:

      • are variables of the primitive data types      

        E.g.: int x, float y, ...

  • Composite (or "structured") variables:

      • are variables that are composed of multiple parts      

  • Examples of structured variables:

      • Array: a series of identically typed variables

      • Object: group of variables of different data types      

General storage characteristics of data structures

  • A data structure (array or linked list) is a composite structure used to store a related information

  • The components in a data structure are stored "close together" in computer memory

    Examples: how arrays and objects are stored in memory:

             

Access expressions for data stored in data structures in high level programming languages

  • High level programming languages can not access all the data inside a data structure all at once

  • High level programming languages always provide access expressions to access one simple variable within the data structure

    Example: access an array element and a field in an object

             

How to accessing a simple variable inside data structures

  • A simple variable within a data structure is located a certain number of bytes after the start location of the data structure:

             

  • The distance from the start of the data structure is called:  

      • offset
