Assembler language programming

What is assembler language programming:

  • Assembler language programming = Writing a computer program in an assembler programming language

  • Assembler programming language = a nmemonic-based programming language where each nmemonic represents (= corresponds) to one computer instruction

  • The nmemonics as usually called assembler instructions

Example asssmbler instructions: 

    add R4, R1, R3  // R4 = R1 + R3
    add R0, R0, R0  // R0 = R0 + R0

Each assembler instruction is translated into exactly one machine instruction

Difference between high level language programming and assembler language programming

Properties of High Level Programming languages:

  • High level programming languages are (1) application oriented and (2) platform (processor) independent

      • Fortran:   scientific applications
      • COBOL:   business applications
      • SIMULA:   computer simulations       

  • Programs written in a high level programming language will run on any type of computer (CPU) (because the compiler will translate the program into the machine language for that computer)

(1) The programmer can focus on designing the algorithm to solve the problem

(2) The programmer do not need to know the computer architecture

Difference between high level language programming and assembler programming

Properties of Assembler Programming languages:

  • Assembler programming languages are computer system oriented and platform (processor) dependent

  • Programs written in an assembler programming language will only run on that specific type of computer (CPU)

  • A programmer writing in an assembler programming language must know the details of the processor (= CPU)
 

Therefore:   before we learn programming in assembler, we must know the details of the CPU
