Question 1. 2's Complement Encoding (20 pts)

    A computer X uses 2-bytes to store the values in this question. Furthermore, the computer uses the 2's complement number system to represent signed numbers.

    Each question is 5 pts.

      1. What are the 2's complement representation for the values 3791(10) and −3791(10) in computer X ?

      2. What decimal values are represented by the following 16 bits 2s complement codes in computer X ? 
                 0000010100101001     ---> value represented = ?
         1111100110011101     ---> value represented = ?

      3. Give the binary pattern that results from adding the two 2s complement codes in part 2: 
                 0000010100101001
      +  1111100110011101
    ----------------------

        What value (in signed decimal) is represented by the resulting sum ?

      4. Give the binary pattern that results from subtracting the two 2s complement codes in part 2: 
                 1111100110011101
      -  0000010100101001
    ----------------------

        What value (in signed decimal) is represented by the difference ?

Question 2. 10's Complement Encoding (20 pts)

    Consider the 10's complement encoding scheme using 4 digits.

    Each question is 5 pts.

      1. What are the 10's complement representation for the value 4292 and −4292 ?

      2. What value is represented by 7193 ?

      3. You are given the following 10s complement representations: 9675 and 9997.

        Add the representations 9675 and 9997. (Truncate the result to 4 digits !)

        What value is represented by the result of the addition ?

        Is the value represented by the resulting representation equal to the addition of the actual values represented by 9675 and 9997 ?

      4. You are given the following 10's complement representations: 9675 and 9997.

        Multiply the representations 9675 and 9997. (Truncate the result to 4 digits !)

        What value is represented by the result of the multiplication ?

        Is the value represented by the resulting representation equal to the product of the actual values represented by 9675 and 9997 ?

Question 3. Binary Arithmetic (10 pts)

    Do the following arithmetic computations in binary.

    1. 11010101 x 1011

      Show the complete long multiplication (i.e.: show the all the multiplication steps) to get full credit

    2. 11010101 / 1011

      Show the complete long division (i.e., show the all the division steps) to get full credit

Question 4. Base-5 Arithmetic (10 pts)

    The following questions are on the number system based on the base number 5.

    • What decimal value is represented by the Base-5 number 34(5) ? (2 pts)
    • What decimal value is represented by the Base-5 number 42(5) ? (2 pts)

      You are given the following addition table for the number system based on the base number 5: 

            + |  1 |  2 |  3 |  4 
    ----+----+----+----+----       
      1 |  2 |  3 |  4 | 10 
    ----+----+----+----+----
      2 |  3 |  4 | 10 | 11 
    ----+----+----+----+----
      3 |  4 | 10 | 11 | 12 
    ----+----+----+----+----
      4 | 10 | 11 | 12 | 13
    • Perform the following computation in base-5 arithmetic using the above addition table: (4 pts) 

      
            34(5) 
          + 42(5)
        ---------

    • What decimal value is represented by the Base-5 number about result in the last question ? (2 pts)

Question 5. Octal, Hexadecimal and Other Numbers (20 pts)

    1. Give the representation of the value 4359 in the octal number system (that is based on the number 8).

    2. Give the representation of the value 4359 in the hexadecimal number system (that is based on the number 16).

    3. Give the representation of the value 4359 in the "nano number system" (that is based on the number 9).

    4. Give the representation of the value 4359 in the "13 number system" (that is based on the number 13; use A to represent 10, B for 11 and C for 12).

Question 6. Encoding (20 pts)

    Show the binary representation (in bits) for the following data items when they are stored in computer memory:

      1. The octal number 1234 in 16 bits
      2. The string "a=34;"
      3. The pattern stored in a byte typed variable a by the Java statement a=34;
      4. The single precision float point number 16.375 in IEEE 754 representation in binary
      5. What is the float point number (written in decimal) represented by the following IEEE 754 representation:
                  11000001111001000000000000000000
