  • Review: Inverse of a matrix (from Math221 - Linear Algebra)

    • Inverse of a matrix (Linear Algebra): 

           Given matrix A

   A-1 is the inverse of matrix A if and only if: 

           A*A-1 = I

   where I = the identity matrix

      Example: 

              +-          -+
      |  7   2   1 |
  A = |  0   3  -1 |
      | -3   4  -2 |
      +-          -+


        
  The inverse of A is:

      +-          -+
      | -2   8  -5 |
  B = |  3 -11   7 |
      |  9 -34  21 |
      +-          -+

 Because:

         +-          -+   +-          -+
         |  7   2   1 |   | -2   8  -5 |
   A*B = |  0   3  -1 | * |  3 -11   7 |
         | -3   4  -2 |   |  9 -34  21 |
         +-          -+   +-          -+

         +-                                             -+
         | -7*2+2*3+1*9    7*8-2*11-1*34   -7*5+2*7+1*21 |    
       = | -0*2+3*3-1*9    0*8-3*11+1*34   -0*5+3*7-1*21 |
         |  3*2+4*3-2*9   -3*8-4*11+2*34    3*5+4*7-2*21 |
         +-                                             -+

         +-       -+
         | 1  0  0 |
       = | 0  1  0 |   <--- identify matrix
         | 0  0  1 |
         +-       -+

  • Review: the Gauss-Jordan elimination algorithm to find the inverse of a matrix

    • In Math221, you have learned the following Gauss-Jordan algorithm to find the inverse of a NxN matrix:

        • Gauss-Jordan Method to find the inverse of: 

                  +-          -+
      |  7   2   1 |
  A = |  0   3  -1 |
      | -3   4  -2 |
      +-          -+

        • We first setup the matrix A,I: 

                      +-          -+ +-          -+
          |  7   2   1 | |  1   0   0 |
  A , I = |  0   3  -1 | |  0   1   0 |
          | -3   4  -2 | |  0   0   1 |
          +-          -+ +-          -+        

  Simplified representation:

     7.000   2.000   1.000  |  1.000   0.000   0.000 
     0.000   3.000  -1.000  |  0.000   1.000   0.000 
    -3.000   4.000  -2.000  |  0.000   0.000   1.000

              Multiply row0 by 1/A[0][0];   // Normalize row 0 of matrix;
