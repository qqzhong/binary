3. The Linked Lists

    The linked lists that I hand coded is contained in the variable section of the hw6.s file. The variable section of the skeletal program contains: 

      
   List list1;		// list1 points to the first list element of list1
			 (it contains the address of the first element of list 1)

   List list2;		// list2 points to the first list element of list2
			 (it contains the address of the first element of list 2)

   List list3;		// list3 points to the first list element of list3
			 (it contains the address of the first element of list 3)

    These 3 linked lists have already been defined so you must have faith in my ability to construct the linked list. You just have to traverse and manipulate the lists. You do not need to create list elements in this assignment (it will happen later) - just manipulate the list elements.

    Each objects in the linked list has the following structure: 

            +-------------+
        | int value   |   (4 bytes)
        +-------------+
        | List next   |   (4 bytes)
        +-------------+
    In Java, the class definition of the List object would look like this: 
    	public class List
	{
	   int  value;
	   List next;
	}

5. Questions

    The skeletal program has 5 sections, labeled as

    • Q1,
    • Q2,
    • ...,
    • Q5

    The labels corresponds to questions 1 through 5.

      Write the assembler instructions the each answer in the corresponding section of the hw6.s file.

    The assignment is as follows:

    1. After the label Q1, write M68000 assembler instructions to do: 
          answer = list1.value ;
      The correct answer is: 1234

    2. After the label Q1, write M68000 assembler instructions to do: 
          answer = list1.next.next.next.value ;

      The correct answer is: 2345

    3. Insert elem1 at the start of list1

        You will see list1 change and the first element is: 
            (7777, 314) -> ... etc

    4. Insert elem2 as the third element in list2

        You will see list2 change and the 3rd element is: 
             ... -> ... -> (8888, 314) -> ... etc

    5. Insert elem3 at the tail (end) of list3
        You will see list3 change and the last element is: 
             ... -> ... -> ... -> (9999, 314)

      NOTE: You must use a while loop to determine the end of the list in Part 5. Do not look in the file and count the number of element in list3. (So do not cheat on yourself by determining the last element in the list through counting the number of elements....) You will lose out on an opportunity to re-inforce your knowledge of linked list (and I will test your ability to work with linked lists in the tests, so you will only be cheating on yourself)
