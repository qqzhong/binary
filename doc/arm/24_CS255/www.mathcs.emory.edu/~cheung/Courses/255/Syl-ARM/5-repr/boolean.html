Representing boolean values inside the computer

  • The (all possible) Boolean values

    • I start with representing the Boolean values because it is easy and intuitive

    • There are only 2 Boolean values: 

           true
   false

  • Representing the Boolean values

    • A representation is a encoding system that allow you to associate a set of values uniquely using the encoding system

    • When we need to represent the Boolean values inside a computer, we must make a association between these 2 sets of values: 

           Boolean values            Values stored by the computer
  --------------------      ---------------------------------
      false                       00000000
      true 			  00000001
				  00000010
				  00000011
				  ...

      The association that many (if not all) computer programming languages use is the following: 

           Boolean values            Values stored by the computer
  --------------------      ---------------------------------    
      false       <-------->      00000000
      true 	  <-------->      00000001
      true	  <-------->	  00000010
      true	  <-------->	  00000011
      ...			  ...

      I.e.: the Boolean value false is represented by the binary number 00000000.

      The Boolean value true is associated with all non-zero binary numbers

  • Example of Boolean representation in the Python programming language

    • I can illustrate the above Boolean representation method using the Python programming language (because Python uses the above method to represent Boolean values).

           Boolean values            Values in Python

      true	  <-------->	  00000010   (= 2)
      true	  <-------->	  00000011   (= 3)
      true	  <-------->	  00000100   (= 4)
      ...			  ...
