The reference data type and reference variables

  • What make C special ?

    • Historical fact:

        • C was designed by Kernighan and Ritchie to help them write (= implement) the UNIX operating system

        • Even today, the UNIX operating system is written (mostly, because some modules must be written in assembler code) in the C programming language

    • What make the C programming language so special:

        • C has a very dangerous data type:

            • The reference data type        

          that no other (popular) programming language has (as far as I can tell).

      The reference data type allows the program reference (= point) to any where in the computer memory and access data stored at that memory location !!!

  • Reference data type

    • Reference:

        • Reference = an address in memory

            • A reference is also known as a pointer

              Example:

              The address 4 points to memory location 4 in the memory

    • Reference (or pointer) data type:

        • Reference data type = a data type that consists of all address of the memory

      Consequently:

        • A variable of the reference data type can store an address of the memory

          (And C has operators that manipulate this address !!!!

          Will be discussed later)

  • Reference variables.... or pointer variables (or "pointers")

    • Reference variable:

        • Reference variable = a variable that can contain a reference (= address)

        • Typically:

            • Reference variable = a variable that contains the address of another variable

      A reference variable is often called a pointer variable

      I will often use both terminology interchangeably !!!

  • Defining a reference variable

    • Syntax: 

             dataType   *varName  ;

      Meaning:

        • The variable varName can contain the address of a variable of the data type dataType

      Important note:

        • The symbol * is also used a the multiply operator

            • The * multiply operator must be used in a statement !!!

        • When the C compiler see the * symbol used in a variable definition, it know that this * symbol is used to define an address typed variable !!!

    • Reference variable:

        • We call a variable that holds an address value: a reference variable

    • Example: 

             int   *p ;     // p can contain the address of an int typed variable   

     float *q ;     // q can contain the address of a float typed variable

      So p and q are reference variables.

      Note:

        • Because p and q points to different types of variables (p points to an int typed variable while q points to a float type variable, the data types of p and q are different

  • Mixed data type definition

    • Important note:

        • You will often see C programs with mixed data type variable definitions.

          Example: the following variable definition 

                 int  a, *p;

          2 variables: 

               Variable   a    is an   int typed            variable

   Variable   p    is an   int reference  typed variable

          So the single definition will define variables of different data types !!!
