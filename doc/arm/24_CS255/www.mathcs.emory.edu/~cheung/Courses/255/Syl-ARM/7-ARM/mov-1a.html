Moving/copying an integer value from a register into another register

  • The mov instruction - again

    • The mov instruction can also be used to move/copy an integer (= 32 bits) value from one register to another register

    • The assembler syntax to move the value stored in a register rM to a register rN (any register in the ARM processor) is: 

            mov rN, rM

      The entire register rM will by copied into the register rN

      Examples: 

            mov  r0, r4      // Copies r4 into r0

    mov  r1, r9      // Copies r9 into r1
