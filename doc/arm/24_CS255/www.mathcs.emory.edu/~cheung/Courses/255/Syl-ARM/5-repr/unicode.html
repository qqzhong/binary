Unicode

  • Advances in computers and computer networks

    • Due to the advent and the huge popularity of the World Wide Web, everyone want have his/her own website on the web.

    • When people from non-English speaking country try to set up their websites, they encountered a problem:

        • The ASCII code standard can only represent the English Alphabet !!!       

      Chinese characters, Hebrew letters, Arabic letters, etc, etc cannot be encoded in the webpage

    • So an international committee is setup to define a standard code for all characters used by all languages in the world.

        • This universal character code is called the Unicode

          Wikipedia page: click here

  • The new Unicode

    • Just to give you an idea what kind of characters the Unicode must represent, here are some of the characters from various languages:

        • English: A, B, C, ...
        • Greek: α, β ...
        • Chinese: 㩝 㩻 ...
        • Korean: 등 릿 ...
        • Arabic: ؤ ب, ...
        • Braile: ⠕ ⡯ ...
        • ....

      Each character must be represented by a unique binary number !!!!

    • Unicode:

        • Unicode = a universal code to represent all symbols used in human communication

        • The unicode uses 16 bits code words

          Therefore:

            • We can represent 216 = 65536 different symbols in Unicode

      You can read more on the Unicode on the Wikipedia website: click here

      There is nothing special about the Unicode. It's just an international standard that all computers use (and agree) on which character corresponds to which binary numbers.
