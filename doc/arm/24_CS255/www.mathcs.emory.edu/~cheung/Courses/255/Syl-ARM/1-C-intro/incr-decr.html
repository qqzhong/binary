The increment operator ++ and the decrement operator --

  • The increment and decrement operators in C

    • Fact:

        • The increment operator (++) and the decrement operator (−−) in C behave the same way as those in Java.

    • Summary of the increment/decrement operators in C 

            operation  | Meaning
  -------------+---------------------------------------------------------   
      x++      | Post increment. Increments x and returns old value of x
      x--      | Post decrement. Decrements x and returns old value of x
               |
      ++x      | Pre increment. Increments x and returns new value of x
      --x      | Pre decrement. Decrements x and returns new value of x
