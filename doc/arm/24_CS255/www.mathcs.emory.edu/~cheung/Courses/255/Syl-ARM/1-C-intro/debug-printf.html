Debugging a C program with "printf( )"

  • Another way to debug (my favorite): use printf( ) between #ifdef DEBUG .... #endif !!!

    • Hint:

        • Insert printf( ) statements in strategic location in your program to print:

            • Variables that can help you understand the execution of your program.

    • After you have identified the crash location, you can add some printf() statements near the crash site and print the relevant variables

      Example: 

            #ifdef  DEBUG
    printf("x = %d\n", x);                 
    printf("y = %d\n", y);
    #endif

      Compile your C program with: 

            gcc  -DDEBUG=    ...

      to enable/include the debug printf( ) statements
