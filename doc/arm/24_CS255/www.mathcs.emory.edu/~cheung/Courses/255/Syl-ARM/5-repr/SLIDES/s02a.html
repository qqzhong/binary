Representing (= Encoding) (all) the Boolean values inside a computer program
 

A commonly used encoding method to represent the Boolean values is as follows: 

   Boolean values            Values stored by the computer
  --------------------      ---------------------------------
      false          <------>     00000000
      true 	     <------>     00000001
      true 	     <------>  	  00000010
      true	     <------>     00000011
				  ...

 

The binary number ZERO (00000000) represents the boolean value false
Any non-zero binary number will represent the boolean value true
