A "trivial" (easy to understand but not practical) conversion function

  • A trivial conversion method to convert string numbers to its 2's complement code

    • You can convert a number string into its 2's complement binary number by testing every possible string input as follows: 

          /* ---------------------------------------------------------------
     Trivial conversion: match the input string to every possible
                         number string and return the corresponding
			 2's complement binary number
     --------------------------------------------------------------- */
  public static int parseInt( String s )            
  {
     if ( s.equals("0") )
        return 0;                // The return statement will return the binary number  
                                 //   00000000000000000000000000000000 !!
     else if ( s.equals("1") )
        return 1;                // returns: 00000000000000000000000000000001
     else if ( s.equals("2") )
        return 2;                // returns: 00000000000000000000000000000010
     ...
     else if ( s.equals("10") )
        return 10;               // returns: 00000000000000000000000000001010
     else if ( s.equals("11") )
        return 11;               // returns: 00000000000000000000000000001011
     ...
     else if ( s.equals("-1") )
        return -1;               // returns: 11111111111111111111111111111111
     else if ( s.equals("-2") )
        1eturn -2;               // returns: 11111111111111111111111111111110    
     ...

  }

    • This method is not practical because you will need to write 232 (= 4294967296) different cases with the if-statements !!!
