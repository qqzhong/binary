ARM's add and subtract instructions
 

The following are the ARM processor's add and subtract instructions: 

 add  rdest, rsrc1, #n    // rdest = rsrc1 + n
 add  rdest, rsrc1, rsrc2  // rdest = rsrc1 + rsrc2

 sub  rdest, rsrc1, #n    // rdest = rsrc1 - n
 sub  rdest, rsrc1, rsrc2  // rdest = rsrc1 - rsrc2

 rsb  rdest, rsrc1, #n    //  rdest = n - rsrc1  (reverse sub)  
 rsb  rdest, rsrc1, rsrc2  //  rdest = rsrc2 - rsrc1   

 n = small constant (between -100 and 100)

ARM's add and subtract instructions - Examples
 

Examples of add and subtract instructions 

 Assembler instruction    Meaning
 ---------------------  -----------------------------
  add  r0, r1, r2         r0 = r1 + r2
  add  r0, r0, r0         r0 = r0 + r0 = 2Å◊r0
  add  r0, r0, #1         r0 = r0 + 1

  sub  r0, r1, r2	  r0 = r1 - r2
  sub  r0, r0, #1	  r0 = r0 - 1

  rsb  r0, r1, r2         r0 = r2 - r1
  rsb  r0, r0, #0         r0 =  0 - r0 = -r0  (negation !!)
 

Comment: neg rm, rn ≡ rsb rm, rn, #0 (short hand)

ARM's multiply instructions

The following is the ARM processor's muliply instruction: 


 mul  rdest, rsrc1, rsrc2 
 
    // rdest = rsrc1 × rsrc2       
    // Important restriction: rdest ≠ rsrc1 !!!
 

Note: ARM does not have a divide instruction

(You need to write a program (algorithm) to perform integer division)

ARM's multiply instructions - Examples
 

Examples of the multiply instruction 

 Assembler instruction    Meaning
 ---------------------  -----------------------------
  mul  r0, r1, r2         r0 = r1 * r2
  mul  r1, r0, r0         r1 = r0 * r0 = r02
 

Comment:   you won't have to use the multiply instruction in cs255 (we don't multiply with large numbers)
