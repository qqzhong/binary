Review: static and dynamic arrays (from CS170 or CS171)

  • Static and dynamic arrays

    • Arrays in programming languages:

        • There are 2 kinds of arrays that programming languages can provide:

            • Static arrays

                • The size (number of elements) associated with the name of the array variable cannot be changed in static arrays

                  (After an array is created, the size of the array cannot be changed)

            • Dynamic arrays

                • The size (number of elements) associated with the name of the array variable can be changed in dynamic arrays

                  (After an array is created, the size of the array can be changed - the array size can grow or shrink)

  • Static arrays

    • Static array:

        • Static arrays are implemented using an (array) identifier

            • The name of the array marks the staring location of the array in memory.

        • The array identifier (= name) is directly associated with the starting location of the array as depicted in the following figure:

  • Advantages and disadvantages of Static and Dynamic arrays

    • Dynamic arrays: (Java !)

        • Advantage: flexibility

        • Disadvantage: slow

            • To access an array element a[i] (assuming you already know the index i), you need at least 2 memory accesses:

              1. One memory access to read the array object reference variable

              2. A second memory access to access the array element itself

    • Static arrays: (Most programming languages)

        • Advantage: speed

            • To access an array element a[i] (assuming you already know the index i), you only need 1 memory access

        • Disadvantage: inflexible

            • You cannot change the size of the (static) array after you have created the array.
