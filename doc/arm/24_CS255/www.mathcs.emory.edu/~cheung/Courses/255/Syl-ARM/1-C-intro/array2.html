Two-dimensional arrays in C

  • Syntax to define a one-dimensional arrays in C

    • Syntax to define an two dimensional array in C: 

           DataType  varName [ size1 ] [ size2 ]  ;

    • Example: 

           double a[5][10];   // Define 5x10 array of double      

   int b[4][20];      // Define 4x20 array of int

    • Example C program: 

           int main( int argc, char* argv[] )                 
   {      
      double a[3][6];    // Define 2 dim array
      int    i, j;
   	  
      for ( i = 0; i < 3; i++ )
   	 for ( j = 0; j < 6; j++ )
   	    a[i][j] = i+j;        // Access element in 2 dim array      
   	  
      for ( i = 0; i < 3; i++ )
      {
   	 for ( j = 0; j < 6; j++ )
   	    printf( "%lf ", a[i][j] );       
     
   	 printf( "\n" );
      }
   }
