Problem: entering integer numbers into the computer

  • Problem description: how to enter numbers into the computer ?

    • Recall that:

        • Internally (= inside a computer program), (signed) integer values are represented using the 2's complement code: 

                 Value            Internal representation (in 32 bits)
   ------------     ----------------------------------------
     ...               ...
     neg three         11111111111111111111111111111101
     neg two           11111111111111111111111111111110
     neg one           11111111111111111111111111111111
     zero              00000000000000000000000000000000
     one	       00000000000000000000000000000001
     two	       00000000000000000000000000000010
     three	       00000000000000000000000000000011
     ...               ...
     twelve            00000000000000000000000000001100
     ...               ...

          The 2's complement representation is very efficient to perform arithmetic calculations

          (See: click here )

    • Numbers are entered into the computer input by typing the text: 

         For example:

     12          
     123   
     ...

      Problem:

        • When you type in the text 

               12

          the computer will receive these ASCII codes: 

                   00110001 00110010                      

 which represents the characters: 1 2

          The computer will not receive the 2;s complement code 00001100 for the value twelve !!!

    • In other words:

        • Human users will enter the number 12 as a string "12" (i.e.: as ASCII codes representing the characters 1 and 2)

        • But internally (= inside a Java program, the boolean values are represented by the binary numbers 00000001 and 00000000

      Therefore: we need to convert the input (ASCII) representation to the internal int (2's complement) representation

    • The Java class Integer has provide the conversion method parseInt( ) to Java programmers to do the conversion.

      In CS170/CS171, you have learned to use this method.

      Example: 

            int i;
    String s = "123";

    i = Integer.parseInt(s);   // Convert string s to integer representation

      In CS255, we will reveal the secret inside the parseInt( ) method so you will understand the computer operation better.
