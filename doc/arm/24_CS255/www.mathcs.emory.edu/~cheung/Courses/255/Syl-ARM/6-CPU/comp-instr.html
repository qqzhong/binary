Intro to representation of computer instructions: What instructions can a computer execute ?

  • What can a computer do ?

    • Recall my mechanical representation of a (electrical) computer:

      A computer can only execute 3 categories (kinds) of instructions:

        1. Data copying operations:

            • Some data is transfered (copied) from a source location to a destination location

              There are 3 ways of data transfer:

                1. Move data: the data is copied from a register to another register inside the CPU
                2. Load data: the data is copied from a memory location to a register inside the CPU
                3. Store data: the data is copied from a register inside the CPU to a memory location

        2. Arithmetic/logical operations:

            • Values from 1 or 2 source (general purpose) registers (inside the CPU) are combined (e.g., negation, add, subtract, multiply, etc) and the result of operation is stored in a destination (general purpose) register

        3. Branch operations: (with or without linking)

            • The Program Counter is updated by the computer instruction

                • Because the CPU will fetch the next instruction from memory at the address given by the value in the Program Counter, updating the Program Counter will cause the CPU to fetch the next instruction at a new location !!!

            • A branch operation will therefore make the program jump to a different part of the program and continues the program execution at the branch address given in the branch instruction

    • Comment:

        • Some computers has instructions that combine 2 (or more) kinds of above basic instructions

          Example:

            • Intel processors can execute arithmetic operations on a value stored in memory

              This type of instruction is performed by combining 3 basic operations:

                1. First, the CPU transfer the data from memory into a scratch register(s) inside the ALU
                2. Then, the CPU performs the Arithmetic/logic operation using values in the scratch register(s) and store the result in (another) register
                3. Finally, the CPU transfer the result in the register to the memory using another data copying operation

          A processer provides combined operations (such as the Intel processor) for the convience of the assembler programmer
